Installation Instructions

  1. Download the extension zip file using the button above and unzip it to a folder on your computer.
  2. Open Chrome and go to chrome://extensions/
  3. Enable Developer mode using the toggle in the top right corner.
  4. Click Load unpacked and select the unzipped LinkSmith folder.
  5. Pin the extension to your toolbar by clicking the puzzle icon and pinning LinkSmith.
